class JiaYuanSpider(builtins.object)
    世纪佳缘爬虫
主要功能为了获取一些女性的数据来测试k-NN算法的精确度
爬虫逻辑是：post登陆，登陆跳转，访问用户主页，访问搜索页面
 
进展：
1. 登陆的借口会封IP
2. 如果大批量分页搜索，会封掉登陆的借口IP
3. 如果大批量的拿验证码图片，会直接拒绝你的IP请求登陆
4. 由于需要获取一些验证码，来训练自己字体库，然后识别
