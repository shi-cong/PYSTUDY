shicong
===========

[![LOGO]][PyPI]

一个代码的二次封装，避免以后做重复的事情，并且为以后的工作大大提高效率的项目。

### 功能：
- amqp 消息队列
- pdf 操作
- ddos 攻击
- 加密
- excel表格 操作
- html xpath re 操作
- json 操作
- mongodb 操作
- mysql 连接池
- numpy 操作
- os 模块 操作
- image 图形操作
- 随机数 操作
- requests 封装
- 字符串 封装
- 深度学习封装
- 时间操作 封装
- tornado 模块封装
- yaml 文件操作 封装

### 环境：
- python版本 >= 3.6

### 安装

    pip install shicong
