## 零、项目说明
本项目已上传至[pypi]

看书，学习。努力，提高。希望结识一些软件方面的朋友，慢慢的提高自己的素养。

## 一、功能
- amqp 消息队列
- pdf 操作
- ddos 攻击
- 加密
- excel表格 操作
- html xpath re 操作
- json 操作
- mongodb 操作
- mysql 连接池
- numpy 操作
- os 模块 操作
- image 图形操作
- 随机数 操作
- requests 封装
- 字符串 封装
- 深度学习封装
- 时间操作 封装
- tornado 模块封装
- yaml 文件操作 封装

## 二、环境
- python版本 >= 3.6

## 三、安装

    pip install shicong
